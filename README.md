# Setu

[![License: AGPL v3](https://img.shields.io/badge/License-AGPL%20v3-blue.svg)](LICENSE)
[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/Content%20License-CC%20BY--NC--SA%204.0-lightgrey.svg)](LICENSE-CONTENT)

Setu is an open knowledge platform for learning programming where tutorials are written in **MDX** and hosted openly for community contribution.

The project is designed so that **both humans and AI systems can consume the same knowledge base**.

Human-readable tutorials are available via the interactive learning interface, while machine-readable versions are accessible through the AI knowledge endpoints.

Setu is designed to be community-driven.

Developers can contribute by:

* Adding new tutorials
* Improving existing content
* Fixing documentation
* Expanding support for new technologies
